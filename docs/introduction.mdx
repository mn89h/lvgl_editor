Build, test, and deploy embedded user interfaces faster with LVGL Pro's integrated suite of development tools. This professional toolkit combines four powerful components designed to streamline your entire UI development workflow.

## Main Tools

<Cards>
  <Card
    icon={<CustomIcon name="xmleditor" />}
    href="./editor/overview"
    title="XML Editor"
  >
    Powerful editor with autocomplete to describe UI components, screens, tests, animations, and data bindings. Features pixel-perfect preview and C code export capabilities.
  </Card>

  <Card
    icon={<CustomIcon name="collaboration" />}
    href="./tools/online-share"
    title="Online Share"
  >
    Web-based version of the Editor that loads XML files from GitHub repositories, enabling easy UI sharing without local development setup.
  </Card>

  <Card
    icon={<CustomIcon name="cli" />}
    href="./tools/cli"
    title="CLI"
  >
    Command-line interface for validating XML files, generating C code, and running UI tests in CI/CD environments.
  </Card>

  <Card
    icon={<CustomIcon name="figma" />}
    href="./tools/figma-integration"
    title="Figma Plugin"
  >
    Extract and synchronize style properties from Figma designs directly into the Editor or via CLI.
  </Card>
</Cards>

## Workflow Benefits

These tools enable developers to:

- **Build** UIs effectively with autocomplete and real-time preview
- **Test** interfaces in a simple and reliable way
- **Collaborate** seamlessly with team members and customers
- **Ship** production-ready code with confidence

This documentation section covers all the details of each tool to help you maximize your productivity with LVGL Pro.
